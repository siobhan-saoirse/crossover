## Source Crossover
The spiritual successor to SourceRoads. This is better because I don't have to deal with SourcePawn jank.
100% human written code. AI is poo poo!!

### How to Install
- Extract the downloaded zip to C:/Program Files (x86)/Steam/steamapps/sourcemods, and remove the -main from the folder name. If there is a single folder in the folder, you need to extrat the folder inside the folder to that directory.
- Install Source SDK 2013 Multiplayer (latest branch)
- Make sure you have an installed copy of Counter-Strike: Source, Half-Life 2: Deathmatch, and Team Fortress 2.
### Reporting a bug
If the game crashed, send me the latest minidump in the dumps directory (C:/Program Files (x86)/Steam/dumps) so I can investigate it and release a quick patch.
